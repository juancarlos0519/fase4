Responda las siguientes preguntas

1. Qué significa CSS

a. Cascading Style Sheets
b. Computer System Standards
c. Creative Style Solutions

2. Qué es una regla CSS

a. Una instrucción que define un estilo para un elemento HTML
b. Una función que realiza una tarea específica en JavaScript
c. Una variable que almacena datos en PHP

3. Cómo se aplica un estilo a un elemento HTML

a. Usando el selector de elementos en una regla CSS
b. Escribiendo el estilo directamente en el atributo "style" del elemento HTML
c. Usando la etiqueta style en el encabezado HTML

4. Qué es un selector

a. Una expresión que identifica los elementos HTML a los que se aplicará un estilo
b. Un tipo especial de variable en JavaScript
c. Un elemento HTML que define una sección de contenido

5. ¿Qué propiedad se utiliza para centrar un elemento horizontalmente?

a. text-align
b. align-center
c. margin: 0 auto;
